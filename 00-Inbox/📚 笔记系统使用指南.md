## 🗂️ 文件夹结构

```
📁 前端笔记仓库/
│
├── 📁 00-Inbox/           # 📥 临时收集箱
│   └── 任何还没整理的东西先放这里
│
├── 📁 10-日记/            # 📔 每日记录
│   └── 用「日记模板」记录每天的生活
│
├── 📁 20-知识笔记/        # 🧠 系统化知识
│   ├── 📁 AI/            # AI 相关学习笔记
│   ├── 📁 前端/          # 前端技术笔记
│   └── 📁 开发/          # 通用开发知识

│
├── 📁 40-摘录/            # ✨ 好句收藏
│   ├── 📁 书籍/
│   ├── 📁 电影/
│   └── 📁 视频/
│

    ├── 知识笔记模板.md

    ├── 摘录模板.md
    └── 感悟模板.md
```

## 📝 使用流程

### 1️⃣ 快速收集 → `00-Inbox`
- 任何想法、链接、截图先扔进来
- 不要纠结分类，先记录再说
- 每周整理一次

### 2️⃣ 定期整理 → 移动到对应文件夹
- 日记 → `10-日记/`
- 知识点 → `20-知识笔记/`
- 灵感 → `30-灵感/`
- 摘录 → `40-摘录/`
- 感悟 → `50-感悟/`

### 3️⃣ 建立连接 → 使用 `[[链接]]`
- 相关笔记之间互相链接
- 形成你的知识网络

---

## 🎯 最佳实践

### ✅ 应该做的
- **频繁记录**：想到什么就记，不要等"完美时机"
- **用自己的话**：不要复制粘贴，要重新组织
- **建立链接**：笔记之间多关联
- **定期回顾**：每周/每月翻看旧笔记
- **积极表扬**：在日记里记录小成就

### ❌ 避免的
- 过度整理（先记录再整理）
- 追求完美（完成比完美重要）
- 只收集不思考（摘录 + 感悟才是完整学习）
